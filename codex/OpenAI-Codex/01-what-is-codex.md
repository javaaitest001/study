# 01. OpenAI Codex 개요 및 특징

## 1. OpenAI Codex 개요
* **정의:** OpenAI에서 개발한 자연어-소스코드 변환 특화 생성형 AI 모델입니다.
* **기반:** GPT-3 모델을 모태로 하되, 자연어 데이터뿐만 아니라 GitHub의 수많은 공개 소스코드 데이터셋을 추가로 학습하여 파인튜닝(Fine-tuning)되었습니다.
* **대표적 응용:** 개발자들이 가장 많이 사용하는 AI 코딩 어시스턴트인 **GitHub Copilot**의 기반 엔진으로 유명합니다.

## 2. 주요 특징
* **코드 생성 및 완성:** 주석으로 기능(예: "이메일 유효성 검사 함수 만들어줘")을 적으면 코드를 자동 생성합니다.
* **리팩토링 및 설명:** 기존 코드를 주고 "이 코드를 가독성 있게 고쳐줘"라거나 "이 코드가 무슨 일을 하는지 설명해줘"라는 요청이 가능합니다.
* **다국어 지원 (Multi-language Support):** 
  - Python에 가장 강력한 성능을 보입니다.
  - JavaScript, TypeScript, Java, C++, Go, PHP, Ruby, SQL 등 대부분의 메이저 프로그래밍 언어를 이해하고 생성할 수 있습니다.
